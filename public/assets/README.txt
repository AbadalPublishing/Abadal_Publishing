Drop these 5 image files into this folder:

1. book-cover.jpg       — Badal: Not Revenge book cover (front)
2. author-waqar.jpg     — Waqar Ali Khan portrait (square works best, ~400×400 or larger)
3. hero-bg.jpg          — Khyber Pass / heritage backdrop for hero section
4. logo.png             — Abadal Publishing wordmark (transparent, white)
5. feather.png          — Feather mark for cursor (transparent)

All files in this folder are served at /assets/<filename> in the app.
For example: /assets/book-cover.jpg → http://localhost:5173/assets/book-cover.jpg

The site already references these paths — once you drop the files in, they'll appear automatically.
